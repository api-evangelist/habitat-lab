# Habitat Lab (habitat-lab)

Habitat Lab is Meta AI Research's embodied AI simulation platform. It provides documentation, quickstart code, and tools for training and evaluating AI agents in photorealistic 3D environments, with Habitat 2.0 focusing on household rearrangement tasks. Habitat Lab is delivered as a Python research library; it does not currently expose a public HTTP API.
